<h1 align="center">Processos em Memória</h1>

<p align="center">Simulador rudimentar que demonstra como é feita a permutação de dados entre memória e disco durante o processamento de dados </p>

 - Compilador
                
       gcc version 9.3.0
 - OS
       
       Ubuntu 20.04.2 LTS
 
  - Executar
 
        terminal:
        ./memoria-simulador/
        ./make clean
        ./make
        ./make run
